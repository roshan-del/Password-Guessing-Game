INSTRUCTIONS:
	1. Download Python (https://www.python.org/downloads/) or make sure you have it downloaded.
	   (IMPORTANT: Check "Add Python # to PATH" at the bottom of setup window)
	
	2. Once downloaded, open command prompt and enter "pip install pillow"
	   (If you encounter any errors visit https://appuals.com/fix-pip-is-not-recognized-as-an-internal-or-external-command/
	    or contact any of our group members)

	3. Make sure game.py, background.jpg, and profile_180.png are all in the same folder/directory.

	4. Navigate to the directory and either click on the game.py file or run it in command prompt as game.py


We hope you enjoy our game and hack your heart out!
